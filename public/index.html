GEMINIVISION

Conversational AI with image understanding capabilities

API Overview

API Key Hidden

The Gemini Vision API provides conversational AI capabilities with image understanding. It supports multi-turn conversations with memory per user ID and can analyze images via URL or base64. Your API key is stored securely on the server and never exposed to the client.

Name
Gemini Vision (Conversational)
Path
GET /api/gemini-vision
Category
AI
Model
gemini-2.5-flash

Parameters

prompt required
Your question or instruction for the AI
uid required
Unique user ID for conversation memory
imgUrl optional
URL of an image to analyze
img optional
Base64 encoded image data

Example Request

GET /api/gemini-vision?prompt=What%20is%20in%20this%20image%3F&uid=user123&imgUrl=https%3A%2F%2Fexample.com%2Fimage.jpg

Example Response

{ "status": true, "response": "This image shows a beautiful sunset over the ocean with palm trees silhouetted against the orange sky.", "conversationId": "user123" }

Special Commands

clear
Send prompt=clear to reset conversation history for a user
